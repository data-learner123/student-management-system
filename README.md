# Student Management System (Python + SQLite)

## 📌 Project Description
This is a simple menu-driven Student Management System built using Python and SQLite. It allows users to perform basic CRUD operations on student records.

---

## 🚀 Features
- Add new student
- View all students
- Search student by name
- Update student marks
- Delete student record

---

## 🛠️ Technologies Used
- Python
- SQLite3 (Database)

---

## ▶️ How to Run
1. Install Python 3
2. Run `main.py`
3. Use menu options in terminal

---

## 📚 Concepts Used
- CRUD operations
- Database connectivity (SQLite)
- Python functions
- User input handling
- Conditional statements & loops
